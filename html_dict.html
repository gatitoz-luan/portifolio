Dicionário de Tags HTML

  • <html>: Define a estrutura básica de uma página HTML.
  • <head>: Contém metadados e links para arquivos externos.
  • <header>: Define o cabeçalho da página.
  • <nav>: Define uma área de navegação com links.
  • <aside>: Define uma barra lateral de informações.
  • <main>: Define o conteúdo principal da página.
  • <footer>: Define o rodapé da página.
  • <section>: Define uma seção de conteúdo dentro de <main>.
  • <ul>: Cria uma lista não ordenada de itens.
  • <li>: Define um item dentro de uma lista.
  • <a>: Cria um link para outra página ou recurso.
  • <img>: Insere uma imagem na página.
  • <p>: Define um parágrafo de texto.
